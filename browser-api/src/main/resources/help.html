Genome Browser

Genome Browser is a desktop and server tool to view different types of biological data.
Description in pdf format can be found on BioLabs group by the link.

Navigation

  • Input form is used to move directly to any genome location. Use chromosome:start-end or gene name to navigate
  • Zoom to region use SHIFT + mouse selection to zoom to any visible region
  • Zoom use SHIFT + mouse scroll, or +/- to zoom in and out
  • Movement use left, right arrow keys for navigation. Also you can use drag and drop on the genome position indicator
  • History use standard actions for your browser to navigate back and forward

Multiple locations

Browser supports displaying multiple locations side by side as a virtual chromosome.
Supported locations (all of these are available in completion popup):
  • Genes
    • All genes available in Ensembl
  • Tss, tes
    • These can be parametrized with width, i.e. TSS500 or boundaries TSS-200,300.
    The same is applicable for TES.
  • Introns, exons, cds, utr5, utr3
  • Repeats, nonRepeats
  • Development genes
  • Housekeeping genes
